Inject labels in to form input elements

Drag-and-drop the button below into your bookmarks bar. Once you have saved this as a bookmark, you can launch it on any web page that has a form that is missing labels for its input elements.

The Javascript code in the bookmarklet will inject labels into the form in an effort to make the forms more accessible.
